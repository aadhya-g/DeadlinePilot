# рҹҡҖ Installation

## Clone Repository

```bash
git clone https://github.com/aadhya-g/DeadlinePilot.git
```

## Move into Project

```bash
cd DeadlinePilot
```

## Install Dependencies

```bash
npm install
```

## Create Environment File

Create

```
.env.local
```

Add

```env
GEMINI_API_KEY=YOUR_GEMINI_API_KEY
```

---

## Start Development Server

```bash
npm run dev
```

Application runs at

```
http://localhost:5173
```
